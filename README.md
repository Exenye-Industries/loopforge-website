# LoopForge Website

Official website for [LoopForge](https://github.com/Exenye-Industries/LoopForge) - Modern Automation Software by Exenye Industries.

## Tech Stack

- **React** + **Vite** - Fast development and optimized builds
- **Tailwind CSS v4** - Utility-first CSS with LoopForge custom theme
- **React Router** - Client-side routing
- **Framer Motion** - Smooth page transitions and animations
- **Lucide React** - Beautiful, consistent icons

## Pages

- **Home** (`/`) - Landing page with hero, features, demo mockup, stats, and CTA
- **Features** (`/features`) - Detailed feature breakdown by category
- **Docs** (`/docs`) - Documentation with sidebar navigation
- **Pricing** (`/pricing`) - Open source pricing with FAQ
- **About** (`/about`) - Vision, values, and tech stack
- **Contact** (`/contact`) - Contact form and links

## Design

The website follows the LoopForge desktop application design language:

- **Dark theme** with deep backgrounds (`#0c0c12`)
- **Purple primary** (`#a855f7`) with teal accent (`#4ecdc4`)
- **Glassmorphism** cards with backdrop blur
- **Inter** font for UI, **JetBrains Mono** for code
- **Framer Motion** animations throughout

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The `dist/` folder contains the production build. It can be deployed to any static hosting service (Cloudflare Pages, Vercel, Netlify, etc.).
